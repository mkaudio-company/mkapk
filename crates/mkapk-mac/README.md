# mkapk-mac

macOS platform backend for [mkapk](https://github.com/mkaudio-company/mkapk): AppKit/`NSView` windowing (a custom `drawRect:`-based paint view, not `lockFocus`), a CoreGraphics render backend for `mkapk-core::PaintCommand`s, CoreText text layout, real mouse input, and an optional Metal GPU surface (`gpu-surface` feature).

See the [workspace README](https://github.com/mkaudio-company/mkapk#readme) for the full picture of how this fits into building VST3/AU/AAX/Standalone plugins.
