# mkapk-widgets

Built-in UI controls for [mkapk](https://github.com/mkaudio-company/mkapk): `Slider`, `Knob`, `Button`, `Label`, and a theme system, built on `mkapk-core`'s widget tree and `mkapk-host`'s parameter gateway.

`#![deny(unsafe_code)]`.

See the [workspace README](https://github.com/mkaudio-company/mkapk#readme) for the full picture of how this fits into building VST3/AU/AAX/Standalone plugins.
