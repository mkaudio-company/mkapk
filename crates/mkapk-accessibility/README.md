# mkapk-accessibility

Accessibility backend trait and platform stubs for [mkapk](https://github.com/mkaudio-company/mkapk), mirroring widget roles/values from `mkapk-core`'s accessibility metadata to real platform accessibility APIs (e.g. `NSAccessibilityElement` on macOS, via `mkapk-mac`).

See the [workspace README](https://github.com/mkaudio-company/mkapk#readme) for the full picture of how this fits into building VST3/AU/AAX/Standalone plugins.
