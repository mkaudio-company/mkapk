# mkapk-win32

Windows platform backend for [mkapk](https://github.com/mkaudio-company/mkapk): Win32 windowing, a Direct2D render backend for `mkapk-core::PaintCommand`s, DirectWrite text layout, real mouse input, and an optional D3D11 GPU surface (`gpu-surface` feature).

See the [workspace README](https://github.com/mkaudio-company/mkapk#readme) for the full picture of how this fits into building VST3/AU/AAX/Standalone plugins.
