# mkapk-test-host

DAW-less standalone test host for [mkapk](https://github.com/mkaudio-company/mkapk): run a plugin's editor for rapid UI iteration without building a real VST3/AU/AAX bundle or opening a DAW.

See the [workspace README](https://github.com/mkaudio-company/mkapk#readme) for the full picture of how this fits into building VST3/AU/AAX/Standalone plugins.
